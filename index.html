• Сортировка пузырьком
  • Данный алгоритм меняет местами два соседних элемента, если первый элемент массива больше второго. Так происходит до тех пор, пока алгоритм не обменяет местами все неотсортированные элементы.
  • Сложность алгоритма O(n²)

  • Сортировка выбором
  • Cуть алгоритма заключается в проходе по массиву от начала до конца в поиске минимального элемента массива и перемещении его в начало.
  • Сложность алгоритма O(n²)
start
  • Сортировка вставками
  • Алгоритм сортирует массив по мере прохождения по его элементам. На каждой итерации берется элемент и сравнивается с каждым элементом в уже отсортированной части массива, таким образом находя «свое место», после чего элемент вставляется на свою позицию. Так происходит до тех пор, пока алгоритм не пройдет по всему массиву. На выходе получим отсортированный массив.
  • Сложность алгоритма O(n²)
